FOREWORD

Ethereals is a "monster catching" game, inspired by Pokémon and Min-Hero: Tower of Sages. I made it as a way to learn Python in a fun way, and you can find the full code on GitHub. This guide was created with the intention to explain the game mechanics, although you're probably familiar with most of it if you've played anything remotely close to Pokémon.

Navigate through the items above to learn more about the universe of Ethereals. I thank you for your attention!
